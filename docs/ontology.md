# Ontology and Core Definitions

This document defines working terminology used within the Telic Coherence (TC) framework. Definitions are provisional and subject to refinement.

## Telic
Refers to behavior or structure oriented toward internally consistent outcomes or goal-directed coherence.

## Coherence
The degree to which a system maintains internal logical consistency across states, decisions, and interactions.

## Agent
Any system capable of modeling its environment and selecting actions based on internal representations.

## Constraint
A limitation or boundary condition that shapes possible system behaviors.

## Observer
A modeling entity that interprets or measures system states through internal representation.

## Alignment
The condition in which agent behavior remains compatible with broader system stability and cooperative outcomes.
